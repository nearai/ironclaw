Flow

Design-system choices

  • Uses current theme tokens: dark neutrals + green accent.
  • Reuses chat messages, approval-card, auth-card, data-card, chips, and tab/sidebar composition.
  • Introduces only one new primitive: the tool-builder card.

Phase-1 scope

Single-user and hosted single-tenant only. Multi-tenant user-authored local installs stay blocked until per-user ToolRegistry isolation exists.
