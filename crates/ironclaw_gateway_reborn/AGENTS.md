# Agent Map — ironclaw_gateway_reborn

## Scope

- Owns Reborn WebChat static assets served at `/v2`.
- Does not own legacy v1 widget/layout customization; that stays in `ironclaw_gateway`.
- Does not own HTTP/API handlers; those stay under `src/channels/web/`.

## Local Notes

- Static app root: `crates/ironclaw_gateway_reborn/static/`.
- Embedded asset table is generated by `build.rs`.
- Dev live filesystem override: `DEV_REBORN_STATIC_DIR=/path/to/static`.
- Gateway root UI selector: `IRONCLAW_GATEWAY_UI=v1|reborn`.
