# Brand token spec

This is the piece other NEAR surfaces (near.com, market.near.ai, Datrix)
can build against so we all stay coherent. Everything below is a `--v2-*`
custom property — the machine-readable catalog is `tokens.ts`, and live
values render under Tokens/Reference.

## What "adopting the system" means

A surface is on-brand when it:

1. imports (or mirrors) the `--v2-*` custom properties from
   `@ironclaw/design-system/tokens.css`,
2. picks colors by role (`--v2-accent`, `--v2-text-muted`), never from a
   palette,
3. uses the structural scales (radius, control density, spacing, type)
   instead of restating values,
4. themes by swapping token values under `[data-theme="…"]` — components
   and layouts stay identical across themes.

## Theming mechanics

- Light values live on `:root`; dark overrides live under
  `:root[data-theme="dark"]`. Structural tokens (radius, spacing, type,
  motion, z) don't change per theme.
- A new surface can re-theme with a single token file and zero component
  changes. The `soft` theme in this Storybook (theme toolbar → *soft*) is
  the proof: warm paper neutrals, espresso ink, a calmer accent, rounder
  radii, softer elevation — all from
  `@ironclaw/design-system/themes/soft.css`.
- The `@theme` block in `tokens.css` maps legacy Tailwind color utilities
  (`--color-iron-*`, `--color-signal`, …) onto the tokens. That's a
  migration bridge for old code, not something new code should use.

## Open brand decisions

Still undecided — current values are good defaults and deliberately easy
to swap:

1. **Typeface.** Geist works but has little flair. Is there an IronClaw
   face (or a pairing) with more character?
2. **Icon library.** Same question for the in-house stroke set + lucide.
3. **IronClaw vs NEAR AI.** Does IronClaw stand as its own brand outside
   near.ai? The answer shapes the accent, the tag language, and how far we
   align with near.com.
4. **How soft is soft?** The *soft* theme shows the Claude-ish direction —
   how far product and marketing surfaces should move is open.
