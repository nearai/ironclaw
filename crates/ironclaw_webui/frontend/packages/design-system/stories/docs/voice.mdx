# Voice & copy

Direction from the product design review: marketing is aligning on a
**softer tone of voice** — human, approachable, plain-spoken — moving away
from edgy/tech-dominant branding. Reference points named in the review:
Claude's product copy and littlebird-style warmth. Nothing here is final
brand language; it is the working guidance until the brand jam lands
decisions.

## Principles

1. **Say what happens, in the user's words.** "Drafts replies for routine
   email" — not "leverages LLM inference for message synthesis."
2. **The agent proposes; the user decides.** Copy around automations always
   preserves agency: *Suggest*, *Review*, *Approve*. Never imply the system
   acted on its own when it didn't.
3. **Calm status language.** Statuses follow the canon: `Success`,
   `Running`, `Warning`, `Failed`, `Paused`. No exclamation marks, no
   alarm-speak; a failed run is a fact, not an emergency.
4. **Sentence case everywhere** — buttons, labels, menu items, tab labels.
   The only uppercase in the system is the pixel-face tag language
   (badges, eyebrows), which is a *typographic* treatment, not a voice.
5. **Short verbs on buttons.** `Save`, `Connect Google`, `Run now`,
   `Delete`. The destructive confirm restates the object ("Delete this
   automation?") and the button repeats the verb, never "OK/Yes".
6. **Empty states teach.** A zero state names the value and the next step
   ("Connect a tool and the agent will start suggesting automations"), not
   just "No data".
7. **Numbers are content.** Stats, durations, and schedules render in
   mono/tabular contexts and stay exact — "128 runs", "every weekday at
   8:00am".

## Built-in component strings

`Modal` (close label) and `ConfirmDialog` (cancel label) render built-in
strings, defaulting to English. Hosts with i18n bridge their translator via
`DesignSystemI18nProvider`; the keys are `common.close` and `common.cancel`.

## Microcopy patterns

| Surface | Pattern | Example |
|---|---|---|
| Primary CTA | verb + object | `Create automation` |
| Confirmation title | question restating the object | `Delete this automation?` |
| Confirmation body | consequence, plainly | `"Morning digest" will stop running immediately. This cannot be undone.` |
| Hint text | why it matters, one sentence | `Shown in the sidebar and page titles.` |
| Error text | what happened + what to do | `This token has expired. Generate a new one in Settings.` |
| Badge label | one word from the status canon | `Running` |
