Send a message as you to a channel or DM. The message appears to come from your account.

Use it when messaging someone or posting somewhere is the task the user asked for. The run's final reply (including routine/trigger results) is delivered to the requesting user automatically; do not use this capability to deliver your reply or a routine/trigger result back to them — it would arrive twice.

The host selects this operation from the capability id. Provide only the parameters described by the input schema; do not include an action field.
