Search or list files and folders.

Use this to find Google Workspace files by name/title, including Google Sheets. For example, to find a spreadsheet named `ABC`, query `name = 'ABC' and mimeType = 'application/vnd.google-apps.spreadsheet' and trashed = false`.

The host selects this operation from the capability id. Provide only the parameters described by the input schema; do not include an action field.
