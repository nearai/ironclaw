How to completely remove T3Claw from your system.

## Stop T3Claw

```bash
# If running in terminal
Ctrl+C

# If running as a service
sudo systemctl stop t3claw   # systemd
brew services stop t3claw    # Homebrew
```

## Remove Binary

<Tabs>
  <Tab title="Linux/macOS">
    ```bash
    # Remove binary
    rm ~/.local/bin/t3claw

    # Or if installed system-wide
    sudo rm /usr/local/bin/t3claw
    ```
  </Tab>

  <Tab title="Ubuntu/Debian (APT)">
    ```bash
    sudo apt remove t3claw
    sudo apt autoremove
    ```
  </Tab>

  <Tab title="macOS (Homebrew)">
    ```bash
    brew uninstall t3claw
    brew untap t3claw-ai/tap
    ```
  </Tab>

  <Tab title="Docker">
    ```bash
    # Stop and remove container
    docker stop t3claw
    docker rm t3claw

    # Remove image
    docker rmi Terminal-3/t3-claw:latest
    ```
  </Tab>
</Tabs>

## Remove Data

<Warning>
This permanently deletes all T3Claw data including conversations, memory, and settings. This cannot be undone.
</Warning>

```bash
# Remove data directory
rm -rf ~/.t3claw

# Remove service files
sudo rm -f /etc/systemd/system/t3claw.service
sudo systemctl daemon-reload

# Remove launchd service (macOS)
rm -f ~/Library/LaunchAgents/ai.t3claw.service.plist

# Remove logs
sudo rm -rf /var/log/t3claw
```

## Remove PostgreSQL Database

If you used PostgreSQL:

```bash
# Remove database (optional)
sudo -u postgres psql <<EOF
DROP DATABASE t3claw;
DROP USER t3claw;
EOF
```

## Verify Removal

```bash
# Check binary removed
which t3claw
# Should return nothing

# Check data removed
ls -la ~/.t3claw
# Should return: No such file or directory

# Check service removed
sudo systemctl status t3claw
# Should return: Unit t3claw.service could not be found
```

## Partial Removal

To keep your data but remove the binary:

```bash
# Just remove binary
rm ~/.local/bin/t3claw

# Data remains at ~/.t3claw/
# Reinstall later and your data will be intact
```
