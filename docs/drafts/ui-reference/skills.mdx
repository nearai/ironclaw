## Skills

View system skills and user-managed skills with their source, trust level, version, and activation metadata. Skills extend IronClaw's capabilities with domain-specific instructions.

<Frame>
  <img src="/assets/screenshots/web-skills-list.png" alt="Installed Skills" />
</Frame>

### UI Elements

- **Skills List**: Displays user-managed, system, and workspace skills with metadata
- **Skill Search**: Search for skills by name or description
- **Add Skill Button**: Opens the skill installation form
- **Edit/Delete Controls**: Available for the signed-in user's manageable skills

### How to Interact

- **Skills List**: Scroll to view all skills grouped by source
- **Skill Search**: Type to filter the skills list in real-time
- **Add Skill Button**: Add a skill from a direct HTTPS URL or pasted SKILL.md content
- **Edit/Delete Controls**: Users can edit their own user-managed skills and delete their own installed skills

### Usage

Use the Skills tab to manage what IronClaw knows. Users can add skills from the registry or direct SKILL.md content, edit their own user-managed skills, delete their own installed skills, or view system skills to understand what's available.
