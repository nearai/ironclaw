# Nuro Agent Interface Pack

This repository now includes a self-contained `nuro_agent/` implementation pack.

## Launch

```bash
bash /Users/nuro/Documents/dev/ironclaw/nuro_agent/scripts/run_nuro_interface.sh
```

Default URL: `http://localhost:8099`

## Contents

- `nuro_agent/interface/` local monitoring UI
- `nuro_agent/config/` safe OpenClaw template
- `nuro_agent/workspace/` private-safe nuro profile and operating files
- `nuro_agent/research/` transcript + synthesis artifacts
- `nuro_agent/scripts/` preflight, refresh, and cron setup scripts
