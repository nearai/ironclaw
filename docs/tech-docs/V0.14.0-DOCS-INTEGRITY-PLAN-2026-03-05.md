# IronClaw v0.14.0 Docs Integrity Plan (2026-03-05)

## Objective

Validate `ironclaw-docs` against released source contracts from `nearai/ironclaw` tag `v0.14.0`, correct factual mismatches, and publish a docs release.

## Scope (released versions only)

- Source baseline: `~/src/ironclaw` at `v0.14.0` (`3e426f0`).
- Docs baseline: `~/src/ironclaw-docs` at `main`.
- Online release metadata:
  - IronClaw latest release API: `https://api.github.com/repos/nearai/ironclaw/releases/latest`
  - Docs latest release API: `https://api.github.com/repos/mudrii/ironclaw-docs/releases/latest`

## Review Method

1. Pull/fetch latest `main` for both repos.
2. Confirm local source equals release tag `v0.14.0`.
3. Compare source contracts to docs:
   - CLI: `src/cli/mod.rs`, `src/cli/tool.rs`
   - Config/env: `src/config/channels.rs`, `src/config/database.rs`, `src/config/llm.rs`
   - Gateway routes: `src/channels/web/server.rs`
4. Run targeted docs grep for stale claims:
   - version examples
   - channel env defaults
   - deprecated/nonexistent env vars
   - missing CLI option coverage
5. Apply minimal factual edits.
6. Commit, tag next docs release, and publish GitHub release entry.
